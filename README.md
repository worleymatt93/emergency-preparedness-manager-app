# Emergency Preparedness Manager

![Platform](https://img.shields.io/badge/platform-Android-green)
![Language](https://img.shields.io/badge/language-Java-orange)
![Build](https://img.shields.io/badge/build-Gradle-blue)

![License](https://img.shields.io/badge/license-GPLv3-blue)

Emergency Preparedness Manager is an Android application designed to help users organize, track, and maintain emergency supply kits. Users can create kits, add supplies, categorize items, and receive notifications when supplies are low or approaching expiration.
